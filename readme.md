# install:

```
npm install
npm run sequelize db:migrate
npm run sequelize db:seed:all
```
you also need to install the Leap Motion SDK : https://developer.leapmotion.com/sdk-leap-motion-controller/
# run
Plug your Leap Motion sensor first
```
npm start
```
or with auto-reload: `npm run dev`

http://localhost:3001/
